# 📅 2025 (NEP UGCF Pattern)

## Section A (Short Answers)

### Q1. Define Cloud Computing
**Answer:**
Cloud computing is the delivery of computing services such as servers, storage, databases, and software over the internet on a pay-as-you-go basis.

---

### Q2. What is Elasticity?
**Answer:**
Elasticity is the ability of a cloud system to automatically increase or decrease resources based on demand.

---

### Q3. Define Virtualization
**Answer:**
Virtualization is the process of creating virtual versions of physical resources like servers or operating systems.

---

### Q4. What is SLA?
**Answer:**
Service Level Agreement (SLA) is a contract between a cloud provider and user defining service quality, uptime, and responsibilities.

---

### Q5. What is Containerization?
**Answer:**
Containerization is a lightweight virtualization method where applications run in isolated environments called containers.

---

## Section B (Medium Answers)

### Q1. Explain Characteristics of Cloud Computing

**Answer:**
1. **On-demand self-service** – Users can access resources without human interaction.  
2. **Broad network access** – Services are accessible over the internet.  
3. **Resource pooling** – Resources are shared among users.  
4. **Rapid elasticity** – Resources can scale quickly.  
5. **Measured service** – Usage is monitored and billed accordingly.  

---

### Q2. Explain IaaS, PaaS, SaaS with Examples

**Answer:**

- **IaaS:** Provides infrastructure like virtual machines and storage.  
- **PaaS:** Provides a platform to develop applications.  
- **SaaS:** Provides software over the internet.

---

### Q3. Differentiate Scalability and Elasticity

| Scalability | Elasticity |
|------------|-----------|
| Increase resources manually | Automatic adjustment |
| Planned growth | Real-time response |

---

## Section C (Long Answers)

### Q4. Explain Cloud Architecture with Diagram

**Answer:**

Cloud architecture consists of:

- **Front-End:** User interface (browser, app)  
- **Back-End:** Servers, storage, database  
- **Network:** Internet connection  

---

### Q5. Explain Security Issues in Cloud Computing

**Answer:**

- Data breaches  
- Data loss  
- Account hijacking  
- Insecure APIs  

**Solutions:**
- Encryption  
- Multi-factor authentication  
- Regular backups  

---

# 📅 2024

## Section A

### Q1. Define Public Cloud
**Answer:**
A public cloud is a cloud service available to the general public over the internet, managed by third-party providers.

---

### Q2. What is MapReduce?
**Answer:**
MapReduce is a programming model used to process large datasets in parallel.

---

### Q3. Define Hypervisor
**Answer:**
A hypervisor is software that creates and manages virtual machines.

---

## Section B

### Q1. Explain Deployment Models

**Answer:**
- Public Cloud – Open to all users  
- Private Cloud – Used by one organization  
- Hybrid Cloud – Combination of both  
- Community Cloud – Shared among organizations  

---

### Q2. Explain Virtualization and its Types

**Answer:**

---

## Section C

### Q3. Explain MapReduce Programming Model

**Answer:**

- **Map Phase:** Processes input into key-value pairs  
- **Reduce Phase:** Combines results  

Example: Word count  

---

### Q4. Advantages and Limitations of Cloud Computing

**Advantages:**

**Limitations:**
