# ☁️ Cloud Computing – Introduction & Topics Overview

## 📌 Introduction

**Cloud Computing** is a technology that allows users to access computing resources (such as servers, storage, databases, networking, and software) over the internet instead of using local machines.

Instead of owning infrastructure, users can **rent resources on-demand**, making it cost-effective, scalable, and flexible.

### 🔑 Key Features
- On-demand self-service
- Scalability and elasticity
- Pay-as-you-go pricing
- Remote accessibility
- High availability

# 📚 Unit-wise Topics Explanation

---

## 🧩 Unit 1: Overview of Computing Paradigm

### 🔹 Introduction to Cloud Computing
- Delivery of computing services via the internet  
- Eliminates need for physical hardware  

### 🔹 Benefits & Limitations

**Benefits:**
- Cost saving  
- Scalability  
- Accessibility  

**Limitations:**
- Security concerns  
- Internet dependency  
- Downtime risks  

### 🔹 History of Computing
- **Mainframes** → Large centralized systems  
- **Cluster Computing** → Multiple systems working together  
- **Grid Computing** → Distributed resources across locations  
- **Distributed Computing** → Tasks shared across systems  

### 🔹 Virtualization
- Creating virtual versions of physical resources (servers, OS)

### 🔹 Service-Oriented & Utility Computing
- **Service-Oriented Computing:** Services provided over network  
- **Utility Computing:** Pay only for what you use  

### 🔹 Cloud Platforms
- Examples: AWS, Azure, GCP  
- Provide infrastructure, tools, and services  

---

## ⚙️ Unit 2: Parallel and Distributed Computing

### 🔹 Parallel Computing
- Multiple processors execute tasks simultaneously  
- Improves speed and performance  

### 🔹 Distributed Computing
- Multiple computers work together over a network  

### 🔹 Differences

| Parallel Computing | Distributed Computing |
|------------------|---------------------|
| Same system | Different systems |
| High speed | Resource sharing |

### 🔹 Architectural Styles
- Client-Server  
- Peer-to-Peer  
- Multi-tier architecture  

---

## 🖥️ Unit 3: Virtualization

### 🔹 Introduction
- Technique to run multiple OS on a single machine  

### 🔹 Characteristics
- Resource sharing  
- Isolation  
- Flexibility  

### 🔹 Virtualization Techniques
- Full virtualization  
- Para-virtualization  
- Hardware-assisted  

### 🔹 Virtualization in Cloud
- Backbone of cloud computing  

### 🔹 Pros & Cons

**Pros:**
- Efficient resource use  
- Cost reduction  

**Cons:**
- Performance overhead  
- Security risks  

### 🔹 Cloud-Enabling Technologies
- **Data Centers:** Physical infrastructure  
- **Containerization:** Lightweight virtualization (e.g., Docker)  

---

## 🏗️ Unit 4: Cloud Computing Architecture

### 🔹 Cloud Architecture
- Structure of cloud components (front-end & back-end)

### 🔹 Cloud Reference Model
- Defines layers and services in cloud systems  

### 🔹 Service Models

- **IaaS (Infrastructure as a Service):**
  - Provides hardware resources (VMs, storage)

- **PaaS (Platform as a Service):**
  - Provides development platform

- **SaaS (Software as a Service):**
  - Ready-to-use software via internet  

### 🔹 Types of Cloud
- Public Cloud  
- Private Cloud  
- Hybrid Cloud

## ✅ Conclusion

Cloud Computing is a **modern computing paradigm** that enables scalable, cost-effective, and efficient use of IT resources. It integrates concepts like:
